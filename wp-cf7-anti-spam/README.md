# CF7 Fortress Anti-Spam

Набор многоуровневой защиты для Contact Form 7:

1. **Honeypot-поле** (невидимое поле, которое заполняют боты).
2. **Time trap** (форма не может быть отправлена слишком быстро).
3. **Rate limiting по IP** (ограничение частоты отправки).
4. **Проверка на избыток ссылок** в сообщении.
5. **Фильтр по стоп-словам**.
6. **Блокировка одноразовых email-доменов**.

## Установка

1. Скопируйте папку `wp-cf7-anti-spam` в `wp-content/plugins/`.
2. Активируйте плагин в админке WordPress.
3. Убедитесь, что Contact Form 7 активен.

Ничего добавлять в шаблон формы CF7 вручную не нужно — поля защиты подставляются автоматически.

## Тюнинг

В файле `wp-cf7-anti-spam.php` можно изменить константы:

- `MIN_FILL_SECONDS`
- `MAX_URLS_IN_MESSAGE`
- `RATE_LIMIT_WINDOW`
- `RATE_LIMIT_ATTEMPTS`
- `DISPOSABLE_DOMAINS`
- `SPAM_KEYWORDS`

## Если при отправке появляется «Произошла ошибка»

1. Включите лог WordPress (`WP_DEBUG_LOG`) и проверьте `wp-content/debug.log`.
2. Проверьте конфликты: временно отключите остальные плагины, кроме CF7 и этого.
3. Обновите CF7 до актуальной версии.
4. Если на сервере нет расширения `mbstring`, новая версия плагина это учитывает (доп. действие не требуется).
5. Если на странице есть JS-конфликты темы/других плагинов, time-trap теперь не блокирует отправку при отсутствующем timestamp.

- Cloudflare Turnstile / Google reCAPTCHA v3

- Логированием spam-срабатываний и авто-блокировкой IP
