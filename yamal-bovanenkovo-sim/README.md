# Yamal Bovanenkovo Construction Simulator

Браузерный симулятор перемещения от первого лица по арктической строительной зоне (Ямал / Бованенково) на чистых HTML/CSS/JS + Three.js через CDN.

## Запуск

### Вариант 1: просто открыть файл
1. Откройте `index.html` двойным кликом в современном браузере.
2. Нажмите **Начать** и разрешите Pointer Lock (при запросе браузера).

### Вариант 2: через локальный сервер (рекомендуется)
```bash
cd yamal-bovanenkovo-sim
python3 -m http.server 8000
```
Откройте `http://localhost:8000`.

## Управление
- `W A S D` — ходьба
- `Shift` — ускорение
- `Space` — легкий прыжок
- `Mouse` — обзор
- Кнопка в HUD — переключение режима освещения и плотности метели
- HUD показывает текущую зону, координаты, режим освещения и тип погоды

## Особенности
- Большая снежная карта с вариативным рельефом
- Центр стройки, модульный городок, трубы, периферия, тундра
- Техника из примитивов: бульдозеры, самосвалы, кран, трубоукладчик, вездеходы
- Погодные эффекты: снег, туман, ветер
- Редкие песцы и очень редкие белые медведи на периферии
- Два режима освещения: дневной/сумеречный
- Два режима освещения: дневной/сумеречный + медленная авто-смена
